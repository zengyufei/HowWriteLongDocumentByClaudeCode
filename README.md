# 调教 LLM 以文件系统组织长文档生成提示词（全领域）

这是一个基于Claude Code的自动化文档生成工具，专门用于生成文档（长文档，字数较多，拆分多文件）。

## 项目概述

本项目通过Claude Code的智能代理系统，自动化生成小型图书管理系统的完整文档体系，包括可行性分析报告和项目愿景与范围文档。

## 功能特点

- **自动化文档生成**：基于AI智能代理自动生成专业的技术文档
- **结构化输出**：生成规范的目录结构和编号体系
- **中文支持**：所有文档内容均为简体中文
- **状态驱动**：采用扫描-填充-重命名循环机制确保文档完整性

## 快速开始

### 环境要求

- Windows操作系统
- Claude Code CLI工具
- 网络连接（用于大模型调用）

### 操作步骤

1. **准备需求文件**
   
   - 编辑 `需求.txt` 文件，填入您的具体需求
   - 或者使用 `完整需求.txt` 中的完整需求

2. **启动Claude Code**
   
   - 双击运行 `启动命令.bat` 文件
   - 等待命令行终端启动并加载Claude Code

3. **执行文档生成**
   
   - 复制 `需求.txt` 中的内容
   - 在Claude Code终端中粘贴需求内容
   - 按回车键开始执行

4. **确认文档大纲**
   
   - Claude Code会首先生成文档大纲结构
   - 查看生成的目录树，确认符合预期
   - 输入"继续"确认大纲结构

5. **自动文档生成**
   
   - 系统会自动创建文件夹结构和文档框架
   - 进入扫描-填充-重命名循环
   - 每个阶段完成后需要输入"继续"继续下一步

### 文件说明

- `CLAUDE.md` - Claude Code的规则文件，包含智能代理的提示词
- `启动命令.bat` - 启动Claude Code的批处理脚本
- `完整需求.txt` - 完整的需求说明文档
- `需求.txt` - 裁剪后的需求，用于快速测试
- `目录.md` - 生成的文档目录索引

## 使用提示

1. **交互式流程**：文档生成过程是交互式的，需要在关键节点输入"继续"
2. **阶段性执行**：系统会分阶段执行，每个阶段完成后暂停等待确认
3. **状态监控**：可以通过查看生成的目录结构了解当前进度

## 生成文档结构

系统将生成以下文档结构：

- `01_可行性分析报告/` - 包含技术、经济、操作等方面的可行性分析
- `02_项目愿景与范围文档/` - 包含项目目标、范围、需求等详细说明

## 注意事项

- 确保网络连接稳定，Claude Code需要调用云端模型
- 文档生成过程可能需要较长时间，请耐心等待
- 生成的文档为Markdown格式，便于后续编辑和使用
- 所有文档内容均为简体中文，符合国内技术文档规范

## 技术支持

如在使用过程中遇到问题，请检查：

1. Claude Code是否正确安装和配置
2. 网络连接是否正常
3. 需求文件格式是否正确
4. 系统资源是否充足
